## ğŸ“¦ Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd furni-furniture-ecommerce
   ```

2. **Install Dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   - Frontend: http://localhost:5173
